# Releases

Diretório padrão para organização de artefatos de release, manifests e índices operacionais.

## Subpastas

- `releases/manifests/` para catálogos, inventários e índices
- `releases/evidence/` para ponteiros para evidências brutas
- `releases/notes/` para notas e pacotes narrativos de release

## Fontes correlatas

- Evidências brutas: `artifacts/release/`
- Processo: `docs/release/release-process.md`
- Runbook: `docs/operational/runbooks/production-release-runbook.md`
